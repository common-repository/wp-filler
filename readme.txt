== Description ==

This plugin lets you insert filler text into any page or post. To use, simply insert the shortcode (like the example below), where paragraphs = the number of filler text paragraphs you want to include.

[wp-filler paragraphs="1"]

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Wp-Filler.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [wp-filler paragraphs="1"] shortcode into your page/post, specifying the number of paragraphs you want.
